## 🐛 Troubleshooting

### Translation Not Working?

**Check:**
1. ✅ Internet connection (Google Translate needs internet)
2. ✅ Transcription text exists
3. ✅ Target language selected
4. ✅ Backend server running

**Common Issues:**
- **"Translation failed"** - Check internet connection
- **Same text returned** - Translation service might be down, try again
- **Weird translation** - Check transcription is correct first

### Translation Quality Issues?

**Tips:**
1. **Improve transcription** - Speak more clearly
2. **Use proper grammar** - Better input = better output
3. **Avoid slang** - Use standard language
4. **Check spelling** - Edit transcription if needed
